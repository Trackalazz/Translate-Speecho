Transcripción de Audio a Texto

Convierte tu voz en texto en tiempo real

Para comenzar a transcribir, necesitamos acceso a tu micrófono. Por favor, haz clic en el botón para habilitar los permisos.

Estado: Inactivo
Idioma: Español

Instrucciones de uso:

  • Haz clic en "Habilitar Micrófono" para conceder permisos
  • Presiona "Iniciar Grabación" para comenzar a transcribir
  • Para detener la grabación, vuelve a presionar el mismo botón (ahora "Detener Grabación")
  • Para añadir signos de puntuación, simplemente dilos en voz alta (punto, coma, etc.)
  • Se añadirá automáticamente un espacio después de cada punto
  • No se usan mayúsculas automáticas a menos que haya un punto previo
  • Usa "Copiar Texto" para copiar el texto transcrito al portapapeles
  • Presiona "Limpiar Todo" para borrar el texto y comenzar de nuevo
